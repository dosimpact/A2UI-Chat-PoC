# CP1: Goal & Non-goal (Scope Baseline)

## 최종 목표 (1줄)
A2UI 채팅-UI 통합 Baseline을 기준으로 `renderers/react`의 공개 API와 카탈로그 초기화/렌더링 규칙을 고정하고, React/Next 소비자 샘플에서 JSON 생성-렌더링-액션 round-trip을 일관되게 검증한다.

## 비목표
- 에이전트/LLM 핵심 모델 구현 또는 외부 모델 벤더 연동 안정화.
- `v0.9/v0.10` 런타임의 새 기능 동시 병행 적용.
- 모바일 네이티브, Electron, 데스크톱(React Native/Flutter) 클라이언트 지원.
- 오픈 소스 릴리즈/문서 사이트 운영(공개 채널 배포 준비).
- 시각회귀/접근성/성능 실측 자동화(운영 수준 CI/CD 완성).

## 우선순위 기준
1. **기초 일치 기준**: `web_core`/`renderers/react`/샘플 간 계약과 카탈로그 동기화(동작 신뢰도)
2. **통합 안정성 기준**: 기본 렌더링 경로와 액션 round-trip 자동화 테스트가 안정적으로 통과되는지
3. **확장성 기준**: shadcn 어댑터 기준을 지키며 새 컴포넌트 온보딩이 예측 가능하게 가능한지
4. **운영 준비 기준**: 문서-코드-체크포인트 간 추적 가능성 확보

## 적용 규칙
- 목표가 변경될 경우 1페이지 요약을 업데이트하고, CP1 산출물(`Plan/assets/01-scope/goal-and-non-goals.md`)과 연동한다.
- CP2 이상 항목은 CP1 목표 고정 상태에서만 진행한다.
