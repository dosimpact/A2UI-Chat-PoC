# Checkpoint 04: React Renderer & Component Adapters

## 목표
- A2UI 컴포넌트 계약을 React 렌더러 + DS 어댑터로 안정적으로 구현한다.

## 구현 체크리스트
- [ ] 공개 API 고정(`A2UIProvider`, `A2UIRenderer`, hooks).
- [ ] 초기화 순서 고정(catalog 등록, style injection, provider wiring).
- [ ] 기본 컴포넌트 팩 구현 우선순위 고정:
  - [ ] Foundation: Text, Button, Card, Row/Column/List
  - [ ] Input: TextField, Checkbox, Slider, DateTime, MultipleChoice
  - [ ] Overlay/Nav: Modal, Tabs, Popover 계열
- [ ] unknown/invalid component fallback 정책 정의.
- [ ] 보안 규칙 정의(markdown sanitize, 외부 URL policy).
- [ ] 런타임 에러 경계 및 recover/fatal 분류 정의.
