# Checkpoint 03: Design System & Theme Tokens

## 목표
- shadcn 구현을 내부 디자인 시스템 토큰으로 제어하고 A2UI theme와 연결한다.

## 구현 체크리스트
- [ ] 토큰 계층 정의(global token / semantic token / component token).

- [ ] A2UI theme -> 내부 token 매핑 테이블 작성.
- [ ] light/dark + 브랜드팩 구조 설계.
- [ ] 컴포넌트에서 하드코딩 스타일 금지 규칙 정의.
- [ ] 토큰 변경 영향분석 절차 정의.
